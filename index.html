Message Board

DEMO

Message Board is a web application designed for real-time, user-generated messaging. It provides a simple platform where registered users can post short messages to a shared feed. Users can create an account, log in securely, and interact with others through a clean and intuitive interface. The application uses PostgreSQL (via Supabase) for data storage, Express.js for backend logic, and Passport.js for authentication. It supports session-based login and features secure cookie handling for cross-origin communication between the React frontend and the Render-hosted backend.
